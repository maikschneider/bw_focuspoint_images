============
Installation
============

..  _requirements:

Requirements
-----------

* TYPO3 v9 LTS or higher

..  _steps:

Installation
------------

Install via Composer

.. code-block:: bash

   composer require blueways/bw-focuspoint-images

Or download it from the
`TYPO3 extension repository <https://extensions.typo3.org/extension/bw_focuspoint_images>`__.
